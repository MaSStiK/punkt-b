# Платформа «Пункт Б»

Коммерческий проект для международной онлайн-школы выбора профессии **«Пункт Б»**.

«Пункт Б» - внутренняя платформа для профориентологов, которая объединяет проведение диагностик, управление клиентской базой и формирование персональных карьерных рекомендаций в едином рабочем пространстве.

Автоматизация ключевых процессов позволила специалистам сократить время на рутинные операции, ускорить обработку результатов и повысить качество работы с клиентами.

## 📌 Кратко о проекте

- **Тип проекта:** Коммерческий
- **Период разработки:** 2023-2025
- **Роль:** Frontend Developer
- **Команда:** Designer, Product Manager, Backend Developer
- **Основные технологии:** JavaScript, jQuery, REST API

## 👨‍💻 Моя роль в проекте

### 🔨 Апрель 2023

- Разработал личный кабинет пользователя и административную панель;
- Спроектировал и реализовал интерфейсы для прохождения, сбора и анализа результатов [диагностик](#diagnostics);
- Реализовал поиск, фильтрацию и удобную навигацию по результатам;
- Разработал мобильную версию платформы;
- Интегрировал платформу с внешними сервисами через REST API;
- Работал в кросс-функциональной команде с дизайнером, менеджером продукта и backend-разработчиком;
- Выполнял Code Review, исправление дефектов и поддержку продукта.

### 🧾 Январь 2024

- Разработал модуль [формирования заключений](#conclusion);
- Реализовал генерацию и экспорт персонализированных PDF-документов.

### ⚙️ Октябрь 2025

- Провёл полную переработку архитектуры проекта;
- Выполнил рефакторинг ключевых модулей для повышения производительности и упрощения дальнейшей поддержки.

## 📈 Результаты

- Сократил время работы специалиста с клиентом на **30%**;
- Ускорил проведение и обработку диагностик в **2 раза** благодаря переработке пользовательского интерфейса;
- Автоматизировал процесс подготовки индивидуальных заключений;
- Повысил стабильность и поддерживаемость платформы после архитектурного рефакторинга.

## 🛠️ Технологии

### Frontend
`HTML` · `CSS` · `JavaScript` · `jQuery` · `Moment.js`

### Интеграции
`REST API`

### Инфраструктура
- Миграция проекта с Tilda на собственный хостинг;
- Поддержка и развитие после переноса.
