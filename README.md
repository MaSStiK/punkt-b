# "Point B" Platform

A commercial project for the international career guidance school **"Point B"**.

The platform is designed for **career consultants** who help clients identify suitable career paths and choose professions that match their interests, skills, and goals.

The platform automates the workflow of career consultants:

- Conducting and analyzing career assessment tests;
- Managing a database of clients and assessment results;
- Generating personalized career reports in PDF format.

## 🧩 Responsibilities

### 🔨 April 2023

- Developed the user dashboard and admin panel;
- Created interfaces for entering, collecting, and displaying results of [career assessment tests](#diagnostics) with filtering and search capabilities;
- Developed the mobile version and integrated with an external database via REST API;
- Collaborated with a designer, product manager, and backend developer;
- Bug fixing and code reviews.

### 🧾 January 2024

- Developed the [report generation page](#conclusion) with PDF export functionality.

### ⚙️ October 2025

- Completely redesigned the architecture and refactored the codebase.

## 📈 Results

- Reduced client processing time by **30%**;
- Doubled the speed of conducting and processing assessments through an improved interface;
- Increased platform stability and maintainability after the architectural refactoring.
