# Point B Career Guidance Platform

A commercial web platform developed for **Point B**, an international online career guidance school.

The platform is designed for career counselors and career guidance specialists who help clients identify their strengths, explore career opportunities, and make informed professional decisions.

By automating key operational processes, the platform enables specialists to focus on delivering personalized guidance while reducing administrative workload.

### Key Features
- Conducting and analyzing career assessment tests;
- Managing client records and assessment results;
- Generating personalized career recommendations and reports in PDF format;
- Managing specialists, permissions, and platform access.

## Responsibilities

### April 2023
- Developed the client portal and administrative dashboard;
- Designed and implemented interfaces for completing, managing, and reviewing [career assessments](#diagnostics);
- Added advanced filtering and search functionality for assessment data;
- Built responsive mobile-friendly interfaces;
- Integrated the platform with external services via REST API;
- Collaborated closely with designers, product managers, and backend developers;
- Performed bug fixing, code reviews, and ongoing maintenance.

### January 2024
- Designed and implemented the [report generation module](#conclusion);
- Developed dynamic PDF export functionality for personalized career guidance reports.

### October 2025
- Led a complete architectural redesign of the platform;
- Refactored legacy code to improve maintainability, scalability, and performance.

## Business Impact

- Reduced specialist-client processing time by **30%**;
- Accelerated assessment completion and result processing by **2x** through improved UX and workflow optimization;
- Increased platform reliability and maintainability through architectural modernization.

## Technology Stack

**Frontend:** `HTML`, `CSS`, `JavaScript`, `jQuery`, `Moment.js`  
**Integration:** `REST API`  
**Infrastructure:** Migration from ~~`Tilda`~~ to a dedicated hosting environment
