# Punkt B Platform

A commercial project developed for **Punkt B**, an international online career guidance school.

Punkt B is an internal platform designed for career consultants, combining career assessments, client management, and personalized career recommendation workflows within a single workspace.

By automating key processes, the platform significantly reduced routine administrative work, accelerated assessment processing, and improved the overall quality of client support.

## 📌 Project Overview

- **Project Type:** Commercial
- **Development Period:** 2023-2025
- **Role:** Frontend Developer
- **Team:** Designer, Product Manager, Backend Developer
- **Core Technologies:** JavaScript, jQuery, REST API

## 👨‍💻 Responsibilities

### 🔨 April 2023

- Developed the user dashboard and administrative panel;
- Designed and implemented interfaces for conducting, collecting, and analyzing [career assessments](#assessment);
- Built search, filtering, and navigation tools for assessment results;
- Developed the mobile version of the platform;
- Integrated external services via REST API;
- Collaborated closely with the designer, product manager, and backend developer;
- Performed code reviews, bug fixing, and ongoing product maintenance.

### 🧾 January 2024

- Developed the [report generation module](#conclusion);
- Implemented personalized PDF report generation and export functionality.

### ⚙️ October 2025

- Led a complete architectural redesign of the platform;
- Refactored core modules to improve performance, scalability, and maintainability.

## 📈 Key Results

- Reduced specialist-client workflow time by **30%**;
- Doubled the speed of assessment completion and result processing through UI improvements;
- Automated the generation of personalized career reports;
- Improved platform stability and maintainability through architectural refactoring.

## 🛠️ Technology Stack

### Frontend
`HTML` · `CSS` · `JavaScript` · `jQuery` · `Moment.js`

### Integrations
`REST API`

### Infrastructure
- Migrated the project from Tilda to a dedicated hosting environment;
- Continued platform support and feature development after migration.
