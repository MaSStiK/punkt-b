# Платформа "Пункт Б"

Коммерческий проект для международной онлайн-школы выбора профессии **"Пункт Б"**.  
Платформа предназначена для **профориентологов**, которые помогают клиентам определиться с карьерным направлением и выбрать подходящую профессию.  

Платформа автоматизирует рабочий процесс профориентологов:  
- Проведение и анализ диагностических тестов;  
- Ведение базы клиентов и результатов;  
- Формирование индивидуальных заключений в формате PDF.

## 🧩 Задачи
### 🔨 Апрель 2023
- Разработка личного кабинета и админ-панели;  
- Разработка интерфейсов для ввода, сбора и отображения результатов [диагностических тестов](#diagnostics) с поддержкой фильтрации и поиска;  
- Разработка мобильной версии, интеграция с внешней базы данных через REST API;  
- Работа в команде (дизайнер, менеджер продукта, backend-разработчик);  
- Bug fixing, Code Review;  

### 🧾 Январь 2024
- Разработка [страницы для создания заключения](#conclusion) и выгрузки его в PDF файле; 

### ⚙️ Октябрь 2025
- Полная переработка архитектуры и рефакторинг кода. 

## 📈 Результаты
- Время работы с клиентом сократилось на **30%**;  
- Проведение и обработка диагностик ускорились **в 2 раза** благодаря новому интерфейсу;  
- Повышена стабильность и удобство платформы после архитектурного рефакторинга.
