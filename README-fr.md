# Plateforme « Point B »

Projet commercial développé pour l'école internationale d'orientation professionnelle **« Point B »**.

La plateforme est destinée aux **conseillers en orientation professionnelle**, qui accompagnent leurs clients dans le choix d'un parcours de carrière et d'une profession adaptée à leurs compétences, intérêts et objectifs.

La plateforme automatise le travail des conseillers :

- Réalisation et analyse de tests d'orientation professionnelle ;
- Gestion de la base de données des clients et des résultats ;
- Génération de rapports personnalisés au format PDF.

## 🧩 Responsabilités

### 🔨 Avril 2023

- Développement de l'espace utilisateur et du panneau d'administration ;
- Création des interfaces de saisie, de collecte et d'affichage des résultats des [tests d'orientation](#diagnostics), avec filtres et recherche ;
- Développement de la version mobile et intégration avec une base de données externe via REST API ;
- Collaboration avec un designer, un chef de produit et un développeur backend ;
- Correction de bugs et revues de code.
